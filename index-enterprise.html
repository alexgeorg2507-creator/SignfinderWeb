Разворачивается в вашем контуре

Не SaaS, куда нужно загружать конфиденциальные договоры. SignFinder ставится на вашем сервере одной командой — в Docker.

  • Данные остаются внутри компании
  • Свой ключ LLM-провайдера — биллинг напрямую
  • Поддержка при внедрении и настройке
  • Обновления по вашему графику
$ git clone signfinder
$ docker compose up -d
