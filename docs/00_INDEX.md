# SignFinder — Индекс проектной документации

> С этого момента ведём. Каждое архитектурное решение — в ADR. Каждый деплой — по GIT_WORKFLOW.

## Правило одно

Любое решение "почему так" — сначала ADR, потом код. Не наоборот.
Любой деплой на prod — только по GIT_WORKFLOW, кнопкой, руками, не агентом.
